Getting Started

Some commands on terminal which I have learned

  1. ls for bash and dir for windows terminal => To list all files in current directory
  2. cd "path" => To change current directory
  3. mkdir "name of new directory" => To make a new directory in current directory
  4. cd .. => To go to previous directory
  5. git checkout "name of branch" => To change branch in git
  6. git status => to know current status of your project
  7. git add . or git add "path" => To mark path for commit or "." to mark all for commit
  8. git commit -m "message" => To commit changes in local repository
  9. git push => To publish your work on website
  10. browser-sync start --server --directory --files "*" => For locally hosting your website

HTML

( Hyper Text Markup Language )

Inserting Image

< img src="link" width="" height="" alt="text" title="text" >

  • Allways specify width and height of image as browser preserves space for image, it is helpful in slow internet connection or when link of image is broken.
  • alt is used to show text in place of image in case image is not loaded.
  • title is used to display tooltip text.
  • value of width and height are in pixels.
  • Alt attribute is necessary with img tag in HTML5.

CSS

( Cascading Style Sheets )

Seclectors

selector {
property: value;
}

  • Element Selector

    Defining CSS for a Element(tag) in HTML

    Example :
    p {
    color:blue;
    }
    Use: <p> --- </p>

  • Class Selector

    Creating a Class for a particular set of design

    Example :
    .blue {
    color:blue;
    }
    Use: <p class="blue"> --- </p>

  • Id Selector

    Defining CSS for a Element of particular id

    Example :
    #name {
    color:blue;
    }
    Use: <p id="name"> --- </p>

  • Grouping Selectors

    Grouping two or more selectors having common attributes

    Example :
    div,.blue {
    color:blue;
    }

  • Combining Selectors

    Combining two or more selectors to specify some elements among Same set of elements

    Example :
    p.blue { =>Only elements with class="blue" will be affcted.
    color:blue;
    }
    Use:
    < p class="blue"> --- </p> , Text color will change to blue.
    < div class="blue"> --- </div> , Text color will remain unaffected.

  • Child Selector

    To select "direct child" of an Element

    Example :
    article>p {
    color:blue;
    }
    Use:
    <article>
    <p> --- </p> => Here text color will change to blue.
    </article>
    <p> --- </p> => Here text color will remain unaffected.
    <article>
    <div><p> --- </p></div> => Here text color will remain unaffected.
    </article>

  • Decendant Selector

    To select all similar child of an Element

    Example :
    article p {
    color:blue;
    }
    Use:
    <article>
    <p> --- </p> => Here text color will change to blue.
    </article>
    <p> --- </p> => Here text color will remain unaffected.
    <article>
    <div><p> --- </p></div> => Here also text color will change to blue.
    </article>

  • Pseudo-class Selector

    It specifies a special state of the selected element(s)

    Example :
    button:hover { => Any button over which the user's pointer is hovering.
    color: blue;
    }

  • Selectors are not limited to element selector

    Few Examples :
    .colored p {
    color:blue;
    }
    aticle>colored{
    color:blue;
    }
    .highlight.mainpoint { => All elements with class highlight as well as mainpoint.
    color:blue;
    background-color:green;
    }
