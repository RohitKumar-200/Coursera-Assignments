Getting Started

Some commands on terminal which I have learned

  1. ls for bash and dir for windows terminal => To list all files in current directory
  2. cd "path" => To change current directory
  3. mkdir "name of new directory" => To make a new directory in current directory
  4. cd .. => To go to previous directory
  5. git checkout "name of branch" => To change branch in git
  6. git status => to know current status of your project
  7. git add . or git add "path" => To mark path for commit or "." to mark all for commit
  8. git commit -m "message" => To commit changes in local repository
  9. git push => To publish your work on website
  10. browser-sync start --server --directory --files "*" => For locally hosting your website

Inserting Image

< img src="link" width="" height="" alt="text" title="text" >

  • Allways specify width and height of image as browser preserves space for image, it is helpful in slow internet connection.
  • alt is used to show text in place of image in case image is not loaded.
  • title is used to display tooltip text
  • value of width and height are in pixels
